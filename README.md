# GAI-48: Multimodal Hospitality Creator

## Group 12D4
**Problem Statement:** Implement a multimodal GenAI application for Hospitality concept visualization.

## Tech Stack
- **LLM:** Google Gemini API
- **Image Gen:** Hugging Face Inference API (Stable Diffusion)
- **Vector DB:** FAISS
- **Frontend:** Streamlit

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Add your keys to `.env`.
3. Run the app: `streamlit run app.py`
